## Um verkefnið

Þessi síða inniheldur gamla FabModules forritið ásamt einföldum leiðbeiningum á notkun FabModules með Roland MDX-20.

## Notkun TL:DR

1. Kveikja á server sem keyrir FabModules - sudo npm start
2. Opna FabModules - Fara inn á ip tölu servers
3. Hlaða inn PNG skrá
4. Velja output format
5. Stilla verkfæri og XYZ
6. Senda í Roland MDX-20
